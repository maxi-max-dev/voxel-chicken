# 🐔 老鹰抓小鸡 · 方块版

童年游戏《老鹰抓小鸡》的 3D 网页版，Minecraft 方块像素风。单文件 Three.js，浏览器打开即玩。

**▶️ 在线玩：https://maxi-max-dev.github.io/voxel-chicken/**

- 开局选角：🦅 老鹰 / 🐔 母鸡 / 🐤 队尾小鸡（键盘 1/2/3 或点击），另两个 AI 补位
- WASD / 方向键：移动；空格：技能（老鹰冲刺扑击 / 母鸡张翅当墙 / 小鸡紧急蹲下）
- V：切换视角（跟随 / 第一人称 / 俯视）　·　Tab：手感调参面板
- 移动端：双摇杆 + 单技能大键
- 胜负：老鹰抓满 3 只赢，鸡队撑过 90 秒赢
- M2 里程碑：完整玩法 —— 三角色三技能、母鸡翅膀真墙护娃、链节超载脱手+脱链小鸡回家重连、三套 AI、流程状态机与结算
- 物理内核：链条行为层跟跑 + PBD 约束（M1 调好的手感，未改）
